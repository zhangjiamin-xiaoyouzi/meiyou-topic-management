## 📋 项目描述

这是美柚社区话题管理后台系统的原型实现，包含以下核心功能：

### ✨ 主要功能

- **快速筛选** - 按发帖数阈值快速筛选话题（可配置，默认≥50）
- **批量操作** - 支持全选/取消、批量修改标签、批量下线
- **标签管理** - 4维度标签分类（特色/非特色、长期/短期、高频/低频、官方/非官方）
- **需求说明** - 每个字段都有橙色需求说明按钮，点击可查看详细说明
- **表单验证** - 批量修改时需满足筛选条件（如需改为特色需要先有非特色标签）

## 📝 核心功能说明

### 快速筛选 - 发帖数
- 勾选启用快速筛选
- 输入阈值数字（默认50）
- 表格自动过滤显示发帖数≥阈值的话题
- 点击"需求说明"可查看详细说明

### 批量操作
- 在表格中勾选需要操作的话题
- 批量操作面板会自动显示
- 支持以下操作：
  - **全选当前页**: 一键选中所有可见话题
  - **取消选择**: 清除所有选中
  - **批量修改标签**: 
    - 选择要修改的标签（支持多个维度）
    - 系统会验证话题是否拥有对应的反面标签
    - 仅对符合条件的话题进行修改
  - **批量下线**: 一键下线所有选中话题

### 话题标签
话题标签采用4维度分类体系：

| 维度 | 对立选项 |
|------|----------|
| 热度 | 特色 ↔ 非特色 |
| 周期 | 长期 ↔ 短期 |
| 频率 | 高频 ↔ 低频 |
| 属性 | 官方 ↔ 非官方 |

批量修改时需满足筛选条件：
- 要改为"特色"，该话题必须先有"非特色"标签
- 要改为"长期"，该话题必须先有"短期"标签
- 以此类推

## 📧 需求说明系统

每个关键字段旁都有橙色"需求说明"按钮：
- 点击按钮打开需求说明弹窗
- 弹窗显示该字段的详细说明
- 支持多次打开/关闭
- 点击背景可快速关闭
